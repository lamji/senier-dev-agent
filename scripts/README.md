# RAG API Call Script

This directory contains a script to automate calls to the RAG API as per the rules defined in AGENTS.md.

## Usage

1. Ensure Node.js and axios are installed.
2. Run the script with `node call_rag_api.js`.
3. The script will call the RAG API with a predefined task and log the response.

## Customization

Edit `call_rag_api.js` to change the task or add more functionality as needed.
