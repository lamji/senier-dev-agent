# Event Admin Domain Template

Standard professional dashboard for Event Management (Registry, KPIs, Layout).

## Files
- `index.md`: Main Dashboard View (KPIs + Grid).
- `AdminLayout.md`: High-density Professional Sidenav and Header.
- `useAdmin.md`: ViewModel for data handling.
- `sub-components/BookingTable.md`: Standard high-density data table.

## Identity Standards
- **Font**: Roboto (Mixed weights).
- **Theme**: Dark Sidebar (#1e293b), Light Content (#f8fafc).
- **Interaction**: Strictly Static (No hovers on critical action buttons).
- **Status Dots**: Green (Success), Yellow (Pending), Red (Alert/Cancel).
